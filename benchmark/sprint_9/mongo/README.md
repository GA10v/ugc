# Тест хранилища MongoDB

## Запуск теста
1. Запустить контейнер командой  ```$ docker-compose up -d```
2. Выполнить скрипт в консоле ```$ sh mongo_config.sh```
3. Последовательно выполнить скрипты в файле ```test_mongo.ipynb```
